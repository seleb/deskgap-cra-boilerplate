The UI application failed to load. On Windows, this may be due to a limitation with Edge webviews. There are two workarounds for this issue:

  • Run the application with administrator privileges (not recommended)
  • Add a loopback exception by running the following in a command prompt: 
    CheckNetIsolation LoopbackExempt -a -n="microsoft.win32webviewhost_cw5n1h2txyewy"
